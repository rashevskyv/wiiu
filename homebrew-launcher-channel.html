    Homebrew Launcher (Channel)

    Разделы

    • Инструкция
      • Часть I - Установка каналов

    Мы установим Homebrew Channel таким образом, что при запуске кастомной прошивки, Homebrew Channel появится в виде иконки в системном меню.

    Инструкция

    Часть I - Установка каналов

    1. Запустите Homebrew Launcher
      • Пользователи CBHC должны нажать кнопку во время загрузки, для доступа к меню CBHC и выберете там Homebrew Launcher
      • Пользователи Haxchi должны сначала запустить Haxchi, а затем запустить его ещё раз, удерживая , чтобы войти в Homebrew Launcher
      • Пользователи Mocha CFW должны запустить Homebrew Launcher через браузер
    2. Запустите “WUP Installer GX2”
      • Если “WUP Installer GX2” падает с ошибкой “DSi Exception Has Occurred”, перезапустите приставку и попробуйте заново.
    3. Отметьте галочкой “HBL”
    4. Нажмите “Install” для установки, а затем “Yes” для подтверждения
    5. Выберите “NAND” в качестве места установки
    6. Нажмите , а затем Закрыть программу (Close the software) для выхода из установщика по завершении установки

    Теперь вы можете запускать Homebrew Launcher из кастомной прошивки, просто воспользовавшись Homebrew Channel.
