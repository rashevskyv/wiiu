Запустивши SaveMii WUP Port, ви можете змінити мову додатка, натиснувши кнопку {% include inc/btn.md btn="X" %}.

Виберіть, що саме будете бекапити: 
  * **Wii U Save Management** — дозволяє вибрати конкретні ігри для бекапу Wii U. 
  * **vWii Save Management** — дозволяє вибрати конкретні ігри для бекапу vWii.
  * **Batch Backup**
    * **Backup All** — бекап усіх ігор для vWii та Wii U.
    * **Backup Wii U** — бекап усіх ігор тільки Wii U.
    * **Backup vWii** — бекап усіх ігор тільки для vWii.

Вибравши конкретний тайтл для бекапу, ви можете вибрати слот, у який буде зроблено бекап (**Backup**), або слот, з якого буде відновлено бекап (**Restore**). Також можна стерти дані збережень (**Wipe**). 

Вибравши пакетну обробку, бекап буде зроблено в порожній слот автоматично без підтверджень.

Резервні копії створюються в папці `wiiu/backups` на карті пам'яті.
