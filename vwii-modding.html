    Мы установим Homebrew Channel и другие модификации для встроенного в Wii U эмулятора Wii (vWii - virtual Wii).

    Помните, что без оригинального контроллера Wii, вы не сможете зайти в vWii

    Если вы хотите изменить регион вашей vWii на европейский, перейдите сюда. После выполнения этой инструкции прошивать vWii уже не нужно будет. Вы можете выполнить смену региона вместо прошивки vWii. Результат будет идентичен

    Инструкция

    Часть I - Подготовительные работы

    1. Выключите консоль
    2. Вставьте SD-карту в компьютер
    3. Скопируйте содержимое .zip-архива wiiusdfiles.zip в корень вашей SD-карты
      • Не нужно делать, если уже делали в процессе выполнения руководства
    4. Вставьте SD-карту обратно в консоль
    5. Включите консоль

    Часть II - установка Homebrew Menu

    1. Запустите vWii Compat Installer ()
    2. Нажмите для того, чтобы установить Homebrew Channel
    3. Дождитесь надписи Install succeeded
    4. Нажмите , чтобы вернуться в системное меню
    5. Запустите vWii
      • Если установка прошла успешно, на экране появится Homebrew Channel

    Часть III - установка и настройка Priiloader

    1. Запустите Homebrew Channel
    2. Запустите Priiloader Installer и дождитесь инициализации программы
    3. Нажмите кнопку на контроллере Wii
    4. По завершению работы программы, нажмите
    5. Запустите Load Priiloader
    6. Перейдите в System Menu Hacks и нажмите
    7. Кнопкой переведите в состояние enable следующие пункты:
      • Block Online Updates
      • Wii System Settings via Options Button
      • Remove Deflicker (не трогайте, если используете старый телевизор)
    8. Выберите System settings, чтобы сохранить настройки
    9. Нажмите , чтобы вернуться в главное меню
    10. Выберите Homebrew Channel, чтобы вернуться к выбору Homebrew

    Часть IV - установка cIOS

    Убедитесь, что в корне SD-карты нет папок wad или wads и что нет никаких .wad-файлов нигде, кроме папки /apps/на SD-карте.

    1. Запустите Homebrew Channel на vWii
    2. Запустите d2x cIOS Installer
    3. Установите параметры в верхней части экрана таким образом, чтобы они соответствовали нижеуказанным:
      • Select cIOS : d2x-v11-beta1-vWii
      • Select cIOS base : 56
      • Select cIOS slot : 249
    4. Нажмите для установки
    5. Дождитесь окончания установки и нажмите для продолжения
    6. Установите параметры в верхней части экрана таким образом, чтобы они соответствовали нижеуказанным:
      • Select cIOS : d2x-v11-beta1-vWii
      • Select cIOS base : 57
      • Select cIOS slot : 250
    7. Нажмите для установки
    8. Дождитесь окончания установки и нажмите для продолжения
    9. Установите параметры в верхней части экрана таким образом, чтобы они соответствовали нижеуказанным:
      • Select cIOS : d2x-v11-beta1-vWii
      • Select cIOS base : 58
      • Select cIOS slot : 251
    10. Нажмите для установки
    11. Дождитесь окончания установки и нажмите для выхода

    Часть V - применение патчей к IOS80

    Этот шаг брикнет вашу vWii, если по какой-то причине будет прерван в процессе применения патчей (отключение электроэнергии, например). В таком случае для восстановления будет необходима резервная копи NAND vWii. Убедитесь, что она у вас есть, перед тем как начинать.

    1. Из Homebrew Channel в vWii, запустите “Patched IOS80 Installer for vWii””
    2. Внимательно прочтите предупреждение и подождите 30 секунд, после чего сможете продолжить
    3. Нажмите для установки
      • Все должно пройти очень быстро
    4. По завершении процесса, нажмите любую кнопку, чтобы вернуться в Homebrew Channel

    Часть V - применение патчей к ww-43db

    С помощью этой утилиты применяются патчи для поддержки 16:9 в ряде WiiWare игр

    1. Из Homebrew Channel в vWii, запустите “ww-43-db-patcher””
    2. Дождитесь окончания инициализации патчера
    3. Нажмите нажмите для начала работы программы
    4. Дождитесь окончания работы патчера и нажмите любую кнопку для возврата в Homebrew Channel

    Часть VI - Установка загрузчиков игр vWii и запуск игр

    Об установке игр и лоадеров подробно рассказано на этой странице - Установка игр для vWii

    Список установленных программ в Homebrew Channel:

    • ftpii - FTP-сервер
    • Homebrew Browser 0.3 - возможность загружать новейшие приложения и игры для вашей Wii непосредственно через консоль
    • Riivolution Patcher - инструмент для мгновенного внесения изменений в игры, который работает с розничными дисками
    • YAWM ModMii Edition 1.0.1 - менеджер и установщик WAD-файлов
    • SaveGame Manager GX rev127 - менеджер сохранений
    • WiiXplorer SS 260 - файловый менеджер
    • WiiFlow WFL 5.5.4 - программа для запуска игр с карты памяти
    • USB Loader GX 3.0 r1281 - программа для запуска игр с карты памяти

    Вы можете удалить следующие программы на этом этапе:

    • Priiloader Installer 0.10.0 - это приложение заменяет первую часть системного меню, загружающуюся на Wii. Это позволяет загружать его перед самим меню Wii. Установщик может как установить, так и удалить priiloader
    • Load Priiloader 1.0 - загружает меню Priiloader через Homebrew Loader
    • d2x_cIOS_Installer-vWii.zip - патчи системных модулей
    • Patched IOS80 Installer for vWii 1.0 - устанавливает патченный IOS80 без проверки подписи на vWii
    • ww-43db-patcher 0.3 - обновляет базу данных соотношения сторон 4:3 для WiiWare (43DB) в архиве системного меню vWii U8, заменяя записи. Это позволяет активировать доступ к соотношению сторон 16:9 в WiiWare-играх, где принудительно используется 4:3

    Теперь вы можете установить и использовать любое хомбрю для vWii, например, USB Loader GX, Wiiflow, CFG USB Loader, эмуляторы, и т. д.

    Для того, чтобы научиться устанавливать игры, перейдите сюда

    Внешний жесткий диск необходимо подключать в верхний USB-порт Wii U, иначе vWii не увидит его. Нельзя использовать тот же жесткий диск, что вы используете для игр Wii U. Если вы используете жесткий диск без собственного дополнительного питания, вам понадобится Y-cable.

    Убедитесь, что все .wad-файлы(каналы, форвардеры, игры, и т.д.), которые вы собираетесь устанавливать, совместимы с vWii. Если вы случайно установите .wad-файл не совместимый с vWii, а предназначенный для обычной Wii, вы получите брик вашей vWii и вам придется восстанавливать её из резервной копии NAND.

    Установка кастомных тем в vWii так же приведёт к брику vWii.

    Удаление .wad-файлов вручную без знания тонкостей этого процесса, так же приведут к брику vWii.
