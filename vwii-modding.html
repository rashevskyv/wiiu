    Ми встановимо Homebrew Channel та інші модифікації для вбудованого у Wii U емулятора Wii (vWii - virtual Wii).

    Пам’ятайте, що без оригінального контролера Wii ви не зможете зайти у vWii

    Якщо ви хочете змінити регіон вашої vWii на європейський, перейдіть сюди. Після виконання цієї інструкції прошивати vWii вже не потрібно буде. Ви можете виконати зміну регіону замість прошивки vWii. Результат буде ідентичним

    Інструкція

    Частина I - Підготовчі роботи

    1. Вимкніть консоль
    2. Вставте SD-карту в комп’ютер
    3. Скопіюйте вміст .zip-архіву wiiusdfiles.zip у корінь вашої SD-карти
      • Не потрібно робити, якщо вже робили в процесі виконання посібника
    4. Вставте SD-карту назад у консоль
    5. Увімкніть консоль

    Частина II - встановлення Homebrew Menu

    1. Запустіть vWii Compat Installer ()
    2. Натисніть для того, щоб встановити Homebrew Channel
    3. Дочекайтеся напису Install succeeded
    4. Натисніть , щоб повернутися в системне меню
    5. Запустіть vWii
      • Якщо встановлення пройшло успішно, на екрані з’явиться Homebrew Channel

    Частина III - встановлення та налаштування Priiloader

    1. Запустіть Homebrew Channel
    2. Запустіть Priiloader Installer і дочекайтеся ініціалізації програми
    3. Натисніть кнопку на контролері Wii
    4. По завершенню роботи програми натисніть
    5. Запустіть Load Priiloader
    6. Перейдіть до System Menu Hacks і натисніть
    7. Кнопкою переведіть у стан enable наступні пункти:
      • Block Online Updates
      • Wii System Settings via Options Button
      • Remove Deflicker (не чіпайте, якщо використовуєте старий телевізор)
    8. Виберіть System settings, щоб зберегти налаштування
    9. Натисніть , щоб повернутися в головне меню
    10. Виберіть Homebrew Channel, щоб повернутися до вибору Homebrew

    Частина IV - встановлення cIOS

    Переконайтеся, що в корені SD-карти немає папок wad або wads і що немає жодних .wad-файлів ніде, крім папки /apps/ на SD-карті.

    1. Запустіть Homebrew Channel у vWii
    2. Запустіть d2x cIOS Installer
    3. Встановіть параметри у верхній частині екрана так, щоб вони відповідали вказаним нижче:
      • Select cIOS : d2x-v11-beta1-vWii
      • Select cIOS base : 56
      • Select cIOS slot : 249
    4. Натисніть для встановлення
    5. Дочекайтеся закінчення встановлення та натисніть для продовження
    6. Встановіть параметри у верхній частині екрана так, щоб вони відповідали вказаним нижче:
      • Select cIOS : d2x-v11-beta1-vWii
      • Select cIOS base : 57
      • Select cIOS slot : 250
    7. Натисніть для встановлення
    8. Дочекайтеся закінчення встановлення та натисніть для продовження
    9. Встановіть параметри у верхній частині екрана так, щоб вони відповідали вказаним нижче:
      • Select cIOS : d2x-v11-beta1-vWii
      • Select cIOS base : 58
      • Select cIOS slot : 251
    10. Натисніть для встановлення
    11. Дочекайтеся закінчення встановлення та натисніть для виходу

    Частина V - застосування патчів до IOS80

    Цей крок перетворить вашу vWii на «цеглу» (brick), якщо з якоїсь причини буде перерваний у процесі застосування патчів (наприклад, відключення електроенергії). У такому разі для відновлення буде необхідна резервна копія NAND vWii. Переконайтеся, що вона у вас є, перед тим як починати.

    1. З Homebrew Channel у vWii запустіть “Patched IOS80 Installer for vWii”
    2. Уважно прочитайте попередження і почекайте 30 секунд, після чого зможете продовжити
    3. Натисніть для встановлення
      • Все має пройти дуже швидко
    4. По завершенні процесу натисніть будь-яку кнопку, щоб повернутися в Homebrew Channel

    Частина V - застосування патчів до ww-43db

    За допомогою цієї утиліти застосовуються патчі для підтримки 16:9 у низці WiiWare ігор

    1. З Homebrew Channel у vWii запустіть “ww-43-db-patcher”
    2. Дочекайтеся закінчення ініціалізації патчера
    3. Натисніть для початку роботи програми
    4. Дочекайтеся закінчення роботи патчера і натисніть будь-яку кнопку для повернення в Homebrew Channel

    Частина VI - Встановлення завантажувачів ігор vWii та запуск ігор

    Про встановлення ігор та лоадерів детально розповів на цій сторінці — Встановлення ігор для vWii

    Список встановлених програм у Homebrew Channel

    • ftpii - FTP-сервер
    • Homebrew Browser 0.3 - можливість завантажувати новітні додатки та ігри для вашої Wii безпосередньо через консоль
    • Riivolution Patcher - інструмент для миттєвого внесення змін в ігри, який працює з роздрібними дисками
    • YAWM ModMii Edition 1.0.1 - менеджер та установник WAD-файлів
    • SaveGame Manager GX rev127 - менеджер збережень
    • WiiXplorer SS 260 - файловий менеджер
    • WiiFlow WFL 5.5.4 - програма для запуску ігор з карти пам’яті
    • USB Loader GX 3.0 r1281 - програма для запуску ігор з карти пам’яті

    Ви можете видалити наступні програми на цьому етапі:

    • Priiloader Installer 0.10.0 - цей додаток замінює першу частину системного меню, що завантажується на Wii. Це дозволяє завантажувати його перед самим меню Wii. Установник може як встановити, так і видалити priiloader
    • Load Priiloader 1.0 - завантажує меню Priiloader через Homebrew Loader
    • d2x_cIOS_Installer-vWii.zip - патчі системних модулів
    • Patched IOS80 Installer for vWii 1.0 - встановлює патчений IOS80 без перевірки підпису на vWii
    • ww-43db-patcher 0.3 - оновлює базу даних співвідношення сторін 4:3 для WiiWare (43DB) в архіві системного меню vWii U8, замінюючи записи. Це дозволяє активувати доступ до співвідношення сторін 16:9 у WiiWare-іграх, де примусово використовується 4:3

    Тепер ви можете встановити та використовувати будь-яке хомбрю для vWii, наприклад, USB Loader GX, Wiiflow, CFG USB Loader, емулятори тощо.

    Для того, щоб навчитися встановлювати ігри, перейдіть сюди

    Зовнішній жорсткий диск необхідно підключати у верхній USB-порт Wii U, інакше vWii не побачить його. Не можна використовувати той самий жорсткий диск, який ви використовуєте для ігор Wii U. Якщо ви використовуєте жорсткий диск без власного додаткового живлення, вам знадобиться Y-cable.

    Переконайтеся, що всі .wad-файли (канали, форвардери, ігри тощо), які ви збираєтеся встановлювати, сумісні з vWii. Якщо ви випадково встановите .wad-файл, не сумісний з vWii, а призначений для звичайної Wii, ви отримаєте брік вашої vWii і вам доведеться відновлювати її з резервної копії NAND.

    Встановлення кастомних тем у vWii так само призведе до бріку vWii.

    Видалення .wad-файлів вручну без знання тонкощів цього процесу так само призведе до бріку vWii.
